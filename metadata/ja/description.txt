インターネット検閲の証拠を収集します。 ネットワークの速度とパフォーマンスを測定します。 世界中の情報管理の透明性を高めるために積極的な役割を果たします。
ooniprobe を実行すると、Web サイトのブロック、ネットワークの速度とパフォーマンス、検閲や監視を担当するシステムがネットワークにあるかどうかを調べます。

これらのテストは、世界中のインターネット検閲を明らかにすることを目指すフリーソフトウェアプロジェクト (The Tor Project の下で) OONI (Open ObservatoryOpen Observatory of Network Interference) によって開発されました。
2012年以来、OONI は 190 以上の国々で数百万のネットワーク測定値を収集し、複数のネットワーク干渉のケースを明らかにしています。 ooniprobe を実行することにより、世界中のインターネット検閲の透明性を高めるのに役立ちます。

▶ インターネット検閲の証拠を収集します
OONI の Web 接続性テストは、DNS の改ざん、TCP/IP ブロック、または透過的な HTTP プロキシによってサイトへのアクセスがブロックされているかどうかを調べるように設計されています。 このテストを実行することにより、インターネット検閲の証拠となるデータを収集し、迂回のヒントが提供されます。

▶ 検閲と監視を担当するシステムを検出します
OONI の HTTP 無効リクエスト行テストは、検閲と監視を担当する可能性のあるシステムの存在を明らかにするためのものです。

▶ ネットワークの速度とパフォーマンスを測定します
OONI のネットワーク診断テスト (NDT) の実装は、あなたの近くの M-Lab サーバーに接続し、ネットワークの速度とパフォーマンスの特徴を示す低レベルの TCP/IP 情報を収集することによって、ネットワークの速度を測定します。

▶ オープンデータ
OONI は、オープンデータにより第三者が独立した研究を行い、OONIの知見を検証し、そして/または他の研究の質問に答えることができるため、収集するすべてのネットワーク測定データを公開します。 このようなデータは、世界中のインターネット検閲の透明性を高めるのにも役立ちます。 すべてのデータは OONI エクスプローラーで公開されています: https://explorer.ooni.torproject.org/

▶ フリーソフトウェア
すべての OONI テストとそのNDT実装は、フリーでオープンソースのソフトウェアに基づいています。 https://github.com/thetorproject/ooniprobe-ios でソースコードを見ることができます。

ご注意ください: ooniprobe を実行することは、ISP のサービス規約に違反したり、あなたの国で法的に疑わしいと思われる可能性があります。 ooniprobe を実行すると、禁止されている可能性のある Web サービスに接続し、Tor などの Web 検閲回避方法を使用します。 OONI プロジェクトは、プローブによって提出されたデータを公開します。IP アドレスやその他の識別情報が含まれる可能性があります。 さらに、あなたが ooniprobe を使用していることは、お使いのコンピュータにアクセスできる人、およびインターネット接続を監視できる人 (雇用主、ISP、政府など) には明らかです。
